## 프로젝트 소개

1. posts 게시판에서 category 별 필터링이 가능합니다.
2. velog 게시판은 velog(https://velog.io/@younngg1012)와 동기화 되어있습니다.
3. posts 폴더에 마크다운 파일을 작성하면, 프로젝트 build 시 html로 파싱됩니다.
